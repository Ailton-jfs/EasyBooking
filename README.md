## Descrição
Este projeto tem como objetivo desenvolver um sistema simples de agendamento, permitindo criar, visualizar e cancelar horários.

O foco principal é aplicar conceitos de Quality Assurance (QA), realizando testes para identificar falhas e propor melhorias no sistema.
